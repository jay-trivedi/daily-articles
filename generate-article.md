## Topic Selection

Pick ONE topic from this weighted pool:
- History (25%) — fascinating events, forgotten chapters, turning points
- Mathematics (25%) — beautiful proofs, unsolved problems, surprising connections
- Technology (25%) — how things work, engineering marvels, pivotal inventions
- Rise and fall of empires (15%) — what made them great, what brought them down
- WILDCARD (10%) — something completely unexpected and mind-blowing. Go crazy. Could be anything: deep sea creatures, linguistics, obscure sports, cosmic phenomena, philosophy paradoxes, culinary history, anything.

Do NOT repeat recent topics — check ~/jay/daily-articles/ for existing articles and pick something fresh.

## Writing Style

- 800-1500 words, substantial and well-researched
- Conversational but informative — like a great podcast or essay, not a textbook
- Use a compelling, specific title (not generic)
- Use concrete details, dates, names, numbers
- End with something thought-provoking

## Steps

1. Check existing articles in ~/jay/daily-articles/ to avoid repeats
2. Research the topic using web search
3. Write the article

5. cd ~/jay/daily-articles && git add the new file && git commit -m "article: <title>" && git push
